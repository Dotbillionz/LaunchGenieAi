# LaunchGenie AI – Startup Launch Assistant

**An AI-powered tool for automated product-launch strategies and startup planning.**

---

## 🎯 Overview

**LaunchGenie** is an intelligent assistant designed to automate and streamline the product launch process for startups and entrepreneurs. This interactive demo generates comprehensive, customized launch plans tailored to your specific region and business concept.

### Key Features

- **Regulatory Compliance**: Automatically generates region-specific permits, licenses, and documentation requirements
- **Task Workflows**: Step-by-step execution plans organized by category (branding, compliance, procurement, hiring, launch)
- **Budget Scenarios**: Three-tier budget planning (Bootstrap, Standard, Premium) with detailed breakdowns
- **Procurement Lists**: Categorized shopping lists with direct links to suppliers and equipment
- **Email Templates**: Ready-to-use outreach templates for landlords, vendors, investors, and suppliers
- **Market Intelligence**: Location-based recommendations and compliance guidance

## 🚀 Quick Start

### Option 1: Open Locally
1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Enter your country, region, and startup idea
4. Click "Generate Launch Plan"

### Option 2: Deploy to Static Hosting
This is a pure static site (HTML/CSS/JavaScript) with no backend dependencies. Deploy to:
- **GitHub Pages**: Push to a repo and enable Pages in settings
- **Vercel**: `vercel --prod` or connect your GitHub repo
- **Netlify**: Drag and drop the folder or connect via Git
- **Any static host**: Upload `index.html` and assets

### Option 3: Use URL Parameters
Prefill the form automatically by adding query parameters:

```
https://your-domain.com/?country=Italy&state=Tuscany&idea=Billiards%20Lounge
```
